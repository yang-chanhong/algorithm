1. AVN 에서 음성 출력 하는 경우
    1-1 버튼 관련
        - 버튼 악의 시험 시 AVN 과부하 방지 방안
        - ETCS 음량버튼 입력 시 음량 조절
            * 이전과 같이 5 개의 레벨만 지원
            * 음량은 AVN에서 조절하고 ETCS 버튼은 음성 ON/OFF 제어
    
    1-2 AVN 음량 0 상태에서 음성 출력 레벨 예외 상황 시 동작
    
    1-3 음성 출력 중복
        - AVN의 안내 멘트와 ETCS 관련 멘트 중복 시 처리 방안

    1-4 부팅 중 음성 출력
        - 기존 부팅 중 출력 Case : 카드 미삽입, 오삽입, 불량카드 / 단말기 미발급, 제조번호 없음
    
    1-5 디버그 메시지만 ETCS에서 출력
        - 안테나 체결 상태, 통신 횟수, 발급 정보 조회 출력 등

    1-6 AVN 출력 중 CAN 연동에 이상 발생 시
        - CAN 연동에 문제가 있음을 표시하는 방법
        - ETCS에서 음성 출력으로 전환
        
    1-7 ETCS 단말기 OFF 상태에서 AVN 동작
        - 단말기 OFF 상태에서 단말기 정보 제공 여부

2. ETCS 에서 음성 출력 하는 경우
    1-1 성우
        - AVN과 같은 성우라도 차이가 있을 수 있음
        - 문구 수정에 불리함
